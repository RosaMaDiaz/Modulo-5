## 📌 Descripción del repositorio

Este repositorio contiene una página web sencilla en HTML que presenta un perfil académico personal.  
Incluye información básica del estudiante como datos personales, materias inscritas, una imagen de perfil y un enlace al sitio web oficial de la universidad.

El objetivo de este proyecto es practicar la estructura básica de HTML y la organización de contenido en una página web.

---

## 📁 Estructura de carpetas

/proyecto
│── index.html
│── rosa 2.jpg

- **index.html** → Archivo principal de la página web.
- **rosa 2.jpg** → Imagen utilizada como foto de perfil.

---

## ▶️ Instrucciones de uso

1. Descargar o clonar el repositorio.
2. Asegurarse de que la imagen **rosa 2.jpg** esté en la misma carpeta que el archivo HTML.
3. Abrir el archivo **index.html** en cualquier navegador web (Chrome, Edge, Firefox, etc.).
4. Visualizar la información del perfil académico.
