## 🎯 Resultado de Aprendizaje
El objetivo es planificar la estructura de una página web aplicando fundamentos de organización de contenidos, jerarquía y buenas prácticas tecnológicas.

### Competencias a Desarrollar:
- Analizar el funcionamiento básico de la Web.
- Definir la estructura de archivos y carpetas de un proyecto web.
- Construir estructuras básicas utilizando etiquetas HTML.
- Aplicar reglas de codificación y buenas prácticas.

## 📚 Fundamentos Teóricos (Resumen)
La guía introduce los conceptos esenciales para el desarrollo frontend:

1. **Estructura HTML5:** Uso de etiquetas semánticas para definir áreas de la página (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`).
2. **Jerarquía de Texto:** Implementación correcta de encabezados (`<h1>` a `<h6>`) y párrafos (`<p>`).
3. **Organización del Proyecto:** Importancia de separar recursos en carpetas (imágenes, estilos, documentos).
4. **Funcionamiento Web:** Conceptos básicos sobre cómo se visualizan los documentos en los navegadores modernos.

## 🏗️ Proyecto Práctico: "Países de Centroamérica"
Consiste en la creación de un sitio informativo sobre la región, con una página individual para cada país (Guatemala, El Salvador, Honduras, Nicaragua, Costa Rica y Panamá).

### Requisitos por Página:
- **Información Histórica:** Resumen breve y datos generales.
- **Identidad Nacional:** Inserción de símbolos patrios (bandera, escudo).
- **Geografía:** Datos de extensión territorial y división política (departamentos/provincias).
- **Turismo:** Listado de lugares destacados con imágenes representativas.
- **Estructura Técnica:** Aplicación obligatoria de etiquetas semánticas de HTML5.

## 📂 Estructura de Archivos Sugerida
- `index.html` (Página de inicio/Menú)
- `guatemala.html`, `el-salvador.html`, etc.
- `/assets/img/` (Carpeta para banderas y paisajes)
- `README.md`
