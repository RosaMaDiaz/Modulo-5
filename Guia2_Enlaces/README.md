## 📚 Fundamentos Teóricos

### 1. La Etiqueta de Enlace `<a>`

La etiqueta principal para crear hipervínculos es `<a>` (anchor). Su función es conectar documentos o recursos.

### 2. Atributos Fundamentales

- **`href`**: Define la ruta, dirección URL o destino del enlace.
- **`target`**: Especifica dónde se abrirá el documento vinculado (ej. `_blank` para una nueva pestaña).
- **`title`**: Añade una descripción emergente cuando el usuario pasa el ratón sobre el enlace.
- **`download`**: Indica que el recurso debe ser descargado en lugar de visualizado.
- **`rel`**: Define la relación entre el documento actual y el vinculado.

### 3. Tipos de Enlaces

- **Enlaces Externos:** Dirigen a sitios web fuera del proyecto actual.
- **Enlaces Locales (Internos):** Vinculan páginas dentro del mismo sitio web.
- **Navegación Interna (Anclas):** Utilizan el atributo `#id` para desplazarse a secciones específicas dentro de la misma página.
- **Enlaces de Correo:** Utilizan el protocolo `mailto:` para abrir el cliente de correo.

### 4. Elementos Multimedia y Estructura

- **Imágenes (`<img>`):** Uso de atributos `src` (fuente), `alt` (texto alternativo), `width` y `height`.
- **Tablas:** Estructuración de datos siguiendo modelos de diseño específicos.
- **Estilos en línea:** Aplicación de CSS básico mediante el atributo `style` (color, alineación, tamaño de fuente).

## 🚀 Requisitos de la Práctica "Mundo Bicicleta"

Para completar la práctica basada en esta guía, se deben integrar:

1. Al menos **2 imágenes** con sus respectivos atributos técnicos.
2. Un mínimo de **3 enlaces externos** y **1 enlace local**.
3. Sistema de **navegación interna** mediante identificadores (#id).
4. Un pie de página (`<footer>`) con información de contacto.
5. Una **tabla informativa** que muestre ejemplos de etiquetas y atributos.
