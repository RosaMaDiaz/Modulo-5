¿Qué es HTML?

HTML (Lenguaje de Marcado de Hipertexto) es el lenguaje de marcado estándar para crear páginas web. Este formulario es un ejemplo de cómo usar diferentes elementos de formulario en HTML, incluyendo campos de texto, selecciones, áreas de texto, medidores y barras de progreso.
