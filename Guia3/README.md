## 📌 Descripción

Este módulo forma parte de la asignatura **Diseño de Estructuras de Página Web** y tiene como objetivo que el participante aprenda a **diseñar y estructurar páginas web educativas** utilizando HTML5.  
Se integran recursos multimedia (audio, video), contenido embebido (YouTube, Google Maps, presentaciones), formularios interactivos y etiquetas semánticas para crear sitios web funcionales, accesibles y orientados al aprendizaje técnico.

---

## 🎯 Resultado de Aprendizaje

Al finalizar esta guía, el estudiante será capaz de:

- Insertar imágenes en diferentes formatos y aplicar buenas prácticas de accesibilidad.
- Integrar audio y video en páginas web con etiquetas y atributos de HTML5.
- Incrustar contenido externo mediante `<iframe>` (YouTube, Google Maps, Google Slides).
- Diseñar tablas básicas y avanzadas para organizar información.
- Crear formularios interactivos con campos de texto, correo, contraseñas, listas desplegables, áreas de texto y botones.
- Aplicar etiquetas semánticas (`<header>`, `<nav>`, `<section>`, `<article>`, `<footer>`, etc.) para estructurar páginas web modernas.

---

## 📂 Contenido del Módulo

1. **Imágenes en páginas web**
   - Formatos: JPG, PNG, GIF, SVG.
   - Etiquetas: `<img>`, `<figure>`, `<figcaption>`.
   - Buenas prácticas: uso de `alt`, optimización de tamaño.

2. **Audio y Video**
   - Etiquetas: `<audio>`, `<video>`.
   - Atributos: `controls`, `autoplay`, `loop`, `muted`.
   - Ejercicios guiados: insertar audio y video locales y externos.

3. **Contenido embebido**
   - Uso de `<iframe>` para YouTube, Google Maps y Google Slides.
   - Configuración de atributos y buenas prácticas.

4. **Tablas en HTML**
   - Estructura básica: `<table>`, `<tr>`, `<td>`, `<th>`.
   - Ejercicios guiados: boletín académico con calificaciones.
   - Tablas avanzadas con combinación de celdas.

5. **Formularios en HTML**
   - Etiquetas: `<form>`, `<input>`, `<label>`, `<select>`, `<textarea>`, `<fieldset>`, `<legend>`.
   - Atributos principales: `required`, `maxlength`, `readonly`, `disabled`.
   - Ejercicios guiados: formularios con menús desplegables, selección múltiple, medidores (`<meter>`) y barras de progreso (`<progress>`).

6. **Estructura Semántica**
   - Etiquetas: `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`.
   - Ejercicio guiado: página web con estructura semántica y estilos básicos.

---

## 🛠️ Actividades de Aprendizaje

- Crear páginas con imágenes y aplicar atributos (`src`, `alt`, `width`, `height`, `title`).
- Insertar audio y video locales y externos.
- Incrustar contenido embebido (YouTube, Google Maps, Slides).
- Diseñar tablas de calificaciones y boletines académicos.
- Construir formularios completos con validaciones.
- Implementar una página web con estructura semántica y estilos CSS básicos.

---

## ✅ Buenas Prácticas

- Usar siempre atributos de accesibilidad (`alt`, `label for`, etc.).
- Optimizar recursos multimedia para mejorar rendimiento.
- Mantener una estructura organizada de carpetas.
- Aplicar etiquetas semánticas para mejorar SEO y accesibilidad.
- Validar formularios con atributos HTML5 antes de usar JavaScript.
