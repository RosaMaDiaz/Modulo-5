## 📌 Descripción del repositorio

Este repositorio contiene una página web desarrollada en HTML sobre los países de Centroamérica.

Cada país cuenta con su propia página donde se presenta información relevante como:

- Historia
- Símbolos patrios
- Lugares turísticos imperdibles
- Datos generales

Además, el proyecto incluye imágenes organizadas en carpetas por país, lo que permite una mejor estructura y organización del contenido multimedia.

El objetivo es aplicar conocimientos de desarrollo web, navegación entre páginas y uso de recursos multimedia.

## ▶️ Instrucciones de uso

1. Descargar o clonar el repositorio.
2. Abrir la carpeta **proyecto3**.
3. Verificar que todas las carpetas de imágenes estén en su lugar.
4. Abrir el archivo principal:
   - `proyecto3_index.html`
5. Navegar entre los diferentes países mediante los enlaces.
