## 📌 Descripción del repositorio

Este repositorio contiene una colección de páginas web desarrolladas en HTML, donde cada archivo representa una obra o sección independiente.

El objetivo del proyecto es practicar la creación de múltiples páginas HTML, organizando contenido y navegando entre diferentes documentos dentro de un mismo proyecto.

**Descripción:**

- `obra1.html - obra10.html` → Páginas secundarias con contenido individual de cada obra o sección.

---

## ▶️ Instrucciones de uso

1. Descargar o clonar el repositorio.
2. Abrir la carpeta **proyecto2**.
3. Ejecutar el archivo principal:
   - Abrir `proyecto2.html` en un navegador web.
4. Navegar entre las diferentes páginas (obras) mediante los enlaces disponibles.

---

## 🌐 Funcionalidades

- Navegación entre múltiples páginas HTML
- Organización de contenido por secciones
- Uso de enlaces internos entre páginas
- Estructura básica de un sitio web
